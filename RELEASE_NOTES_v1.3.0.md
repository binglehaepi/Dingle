# Dingle v1.3.0 Release Notes

**릴리즈 날짜**: 2026년 2월  
**플랫폼**: Windows 10/11 (64비트)

---

## ✨ 새로운 기능

### 🔮 오하아사 운세 개선
- 오하아사 순위에 날짜 표시 추가
- 더블클릭으로 강제 새로고침 기능

### 🎵 음악 임베드 강화
- **Spotify 임베드 개선**: 안정성 향상
- **SoundCloud 임베드 추가**: 음악 재생 지원
- YouTube 임베드 최적화

### 📌 소셜 미디어 임베드
- Pinterest 임베드 개선
- Twitter/Instagram Safe Mode 구현

---

## 🔒 보안 강화

### 개인정보 보호
- ✅ 완전한 로컬 저장 (외부 전송 없음)
- ✅ 개인정보 수집 없음
- ✅ 사용자 추적 없음
- ✅ 보안 감사 완료 (2026-02-01)

### Safe Mode 구현
- Twitter/Instagram 원문 저장 금지
- 공식 임베드 위젯만 사용
- 저작권 안전 정책 적용

## 💻 시스템 요구사항

- **OS**: Windows 10/11 (64비트)
- **RAM**: 4GB 이상 권장
- **디스크**: 200MB 여유 공간
- **인터넷**: 임베드 및 운세 기능용

---

## 📥 설치 방법

1. `Dingle Setup 1.3.0.exe` 다운로드
2. 실행 (Windows Defender 경고 시 "추가 정보" → "실행")
3. 설치 진행
4. 완료!

**자동 업데이트**: 한 번 설치하면 자동으로 업데이트됩니다.

## ⚠️ 알려진 이슈

- **원인**: 코드 서명 인증서 없음 ($300/년)

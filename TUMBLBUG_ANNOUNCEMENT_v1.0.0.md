# Dingle v1.0.0 배포 안내

안녕하세요, 빙글해피 팀입니다.

텀블벅 후원자 여러분께 **Dingle v1.0.0** 정식 버전을 배포하게 되어 기쁘게 알려드립니다.

## 다운로드 방법

### 방법 1: 첨부 파일 다운로드

첨부된 `Dingle_v1.0.0_Tumblbug_Setup.zip` 파일을 다운로드하신 후 압축을 해제하여 사용하실 수 있습니다.

**압축 파일 포함 항목:**
- `Dingle Setup 1.0.0.exe` - 설치 파일
- `설치방법.txt` - 설치 안내

- `자동업데이트_안내.txt` - 업데이트 정보
- `README.txt` - 전체 안내
- `커스텀_스티커_추가_가이드.txt` - 스티커 150종 구매자용

### 방법 2: GitHub Release 다운로드 (권장)

**다운로드 링크:** https://github.com/binglehaepi/Dingle/releases/tag/v1.0.0

- Windows 설치 파일: `Dingle Setup 1.0.0.exe` (82.57 MB)

## 주요 기능

### 다이어리 뷰 모드

- **월간 캘린더:** 한 달 전체를 한눈에 확인
- **주간 다이어리:** 일주일 단위 관리
- **일간 다이어리:** 하루 집중 작성

### 꾸미기 기능

- 사진 업로드 및 자유 배치
- 스티커 추가 (회전, 크기 조절 가능)
- 마스킹테이프 장식
- 드래그 앤 드롭 방식의 직관적인 조작

### 멀티미디어 임베드

- YouTube 영상 임베드
- Spotify 음악 플레이어
- SoundCloud 트랙
- Pinterest 이미지
- Twitter 트윗

### 오하아사 운세

- 매일 업데이트되는 별자리 순위
- 더블클릭으로 수동 새로고침 가능
- 코코넛 캐릭터 디자인

### 데이터 관리

- 자동 저장 기능
- 백업 및 복원 기능
- 로컬 저장 방식 (서버 전송 없음)
- 개인정보 미수집

### 자동 업데이트

- 앱 실행 시 자동 업데이트 확인
- 원클릭 업데이트 지원
- 평생 무료 업데이트 제공

## 설치 방법

1. 위 다운로드 방법 중 하나를 선택하여 설치 파일을 받습니다.
2. `Dingle Setup 1.0.0.exe` 파일을 실행합니다.
3. Windows Defender 경고가 나타날 경우:
   - "추가 정보" 클릭
   - "실행" 클릭
   - (코드 서명 인증서 비용 문제로 경고가 표시되나, 안전한 프로그램입니다)
4. 설치 경로를 선택하고 "설치" 버튼을 클릭합니다.
5. 설치 완료 후 바탕화면 아이콘을 클릭하여 실행합니다.

자세한 설치 방법은 첨부 파일 내 **설치방법.txt**를 참고하시기 바랍니다.

## 스티커 150종 추가 구매 후원자 안내

### 현재 버전(v1.0.0) 사용 방법

받으신 스티커 파일을 "사진으로 추가" 기능을 통해 다이어리에 사용하실 수 있습니다.

**사용 절차:**
1. 스티커 이미지 파일 준비 (PNG/WEBP 형식)
2. Dingle 다이어리 페이지 열기
3. 스티커 파일을 다이어리로 드래그 앤 드롭
4. 크기 조절, 회전, 이동 등 자유롭게 편집

### 향후 업데이트 예정 기능 (v1.1.0)

- 커스텀 스티커 라이브러리
- 스티커 폴더 지정 기능
- 자동 썸네일 생성
- 카테고리별 분류

자세한 내용은 첨부 파일 내 **커스텀_스티커_추가_가이드.txt**를 참고하시기 바랍니다.

## 문제 해결

### Windows Defender 보안 경고

"추가 정보" 클릭 후 "실행"을 선택하시면 됩니다.
(코드 서명 인증서가 없어 보안 경고가 표시되나, 안전한 프로그램입니다)

### 창 위치 문제

v1.0.0에서 창이 작업표시줄 아래에 나타나던 문제가 수정되었습니다.
이제 화면 중앙에 정상적으로 표시됩니다.

### 기타 문제

- 첨부 파일 내 **문제해결_가이드.txt** 참고
- GitHub 이슈 제기: https://github.com/binglehaepi/Dingle/issues

## 개인정보 보호

Dingle은 사용자의 개인정보를 수집하지 않습니다.

- 모든 다이어리 데이터는 사용자의 컴퓨터에만 저장됩니다.
- 서버로 데이터가 전송되지 않습니다.
- 개인정보 수집이 이루어지지 않습니다.
- 오픈소스 프로젝트로 공개되어 있습니다.

**관련 문서:**
- [개인정보 보호 정책](https://github.com/binglehaepi/Dingle/blob/main/PRIVACY_POLICY.md)
- [보안 감사 보고서](https://github.com/binglehaepi/Dingle/blob/main/SECURITY_AUDIT_REPORT.md)

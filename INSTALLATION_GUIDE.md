# 📥 Digital Scrap Diary - 설치 가이드

**버전**: 1.0.0  
**최종 업데이트**: 2025-12-18

---

## 🖥️ 시스템 요구사항

- **운영체제**: macOS 10.14 (Mojave) 이상
- **프로세서**: 
  - Apple Silicon (M1/M2/M3) → `arm64.dmg` 권장
  - Intel Mac → `.dmg` (x64 버전)
- **저장 공간**: 최소 500 MB
- **인터넷**: 
  - ✅ 앱 실행: 인터넷 불필요 (완전 오프라인 작동)
  - ⚠️ SNS 임베드 (트위터/인스타그램): 인터넷 필요
  - ⚠️ 링크 스크랩: 인터넷 필요 (실패 시 자동 Fallback)

---

## 📦 1단계: 다운로드

### 어떤 버전을 다운로드해야 하나요?

**Apple Silicon (M1/M2/M3) Mac**:
```
Digital Scrap Diary-1.0.0-arm64.dmg  ⭐ 권장
```

**Intel Mac**:
```
Digital Scrap Diary-1.0.0.dmg
```

**확인 방법**: 
```
 → About This Mac → Chip
  - "Apple M1/M2/M3" → arm64 버전
  - "Intel Core i5/i7" → x64 버전
```

---

## 🔓 2단계: 설치

### 방법 1: DMG 설치 (권장)

1. **DMG 파일 더블클릭**
   ```
   Digital Scrap Diary-1.0.0-arm64.dmg
   ```

2. **앱을 Applications 폴더로 드래그**
   - DMG 창이 열리면 앱 아이콘을 `Applications` 폴더로 드래그앤드롭

3. **DMG 이젝트**
   - Finder 사이드바에서 DMG를 우클릭 → "Eject"

---

## ⚠️ 3단계: 첫 실행 (중요!)

### ❌ 일반적인 더블클릭은 작동하지 않습니다!

macOS는 인터넷에서 다운로드한 앱을 자동으로 차단합니다.

### ✅ 올바른 방법

#### 옵션 A: 우클릭 → 열기 (권장)

1. **Finder 열기** (`⌘ + Space` → "Finder")
2. **Applications 폴더**로 이동
3. **Digital Scrap Diary 앱을 우클릭** (또는 `Control + 클릭`)
4. **"열기"** 선택
5. 경고창에서 **"열기"** 버튼 클릭

```
⚠️ "확인되지 않은 개발자"라는 메시지가 나타나지만
   이것은 정상입니다. 앱은 안전합니다.
```

#### 옵션 B: 시스템 설정 (권장하지 않음)

1. **시스템 설정** → **개인정보 보호 및 보안**
2. **보안** 섹션 하단에서 **"확인 없이 열기"** 클릭

#### 옵션 C: 터미널 명령어 (고급 사용자)

```bash
xattr -cr "/Applications/Digital Scrap Diary.app"
```

그 다음 더블클릭으로 실행

---

## 🔄 4단계: 이후 실행

**첫 실행 이후**에는 일반적인 더블클릭으로 정상 실행됩니다!

- Dock에 추가
- Spotlight 검색 (`⌘ + Space` → "Digital Scrap Diary")
- Launchpad에서 실행

---

## 📂 5단계: 데이터 위치 확인

앱이 실행되면 자동으로 데이터 폴더를 생성합니다:

```
~/Documents/ScrapDiary/
├── current.json          # 현재 다이어리 데이터
└── backups/              # 자동/수동 백업
    ├── diary-2025-12-18T14-30-00.json
    └── diary-2025-12-18T10-15-30.json
```

**자동 저장**: 편집 후 5초마다 자동 저장됩니다.

---

## 🌐 인터넷 연결 요구사항

### ✅ 오프라인 작동
- 앱 실행
- 로컬 데이터 로드/저장
- 이미지 추가/편집
- 스티커 추가/편집
- 텍스트 편집
- 레이아웃 변경
- 백업/복원

### ⚠️ 인터넷 필요
- **링크 스크랩**: URL 메타데이터 가져오기
  - 실패 시: 자동으로 링크 카드 Fallback 생성
- **트위터/인스타그램 임베드**: 공식 위젯 로드
  - 실패 시: 링크 카드로 대체 표시

**권장**: 링크 스크랩은 인터넷 연결 시 미리 추가

## 📤 내보내기 권장사항

### 🛡️ 안전 모드 (권장)

PNG/PDF 내보내기 시 **"안전 모드"** 활성화를 권장합니다:

```
✅ SNS 임베드 제외 (저작권 안전)
✅ 링크 카드만 포함
✅ 블로그/SNS 공유에 적합
```

**사용 방법**:
1. PNG/PDF 버튼 클릭
2. "안전 모드" 체크박스 활성화
3. 내보내기 실행

### 📷 일반 모드

```
⚠️ SNS 임베드 포함 (개인 보관용)
⚠️ 공개 공유 시 저작권 주의
```

## 🐛 문제 해결

### Q1. 앱이 실행되지 않습니다 (더블클릭 시)

**A**: 3단계 "첫 실행" 섹션을 다시 확인하세요.
- 반드시 **우클릭 → 열기**로 첫 실행 필요

---

### Q2. "손상되어 열 수 없습니다" 오류

**A**: 터미널에서 다음 명령어 실행:

```bash
xattr -cr "/Applications/Digital Scrap Diary.app"
```

그 다음 다시 실행

---

### Q3. 회사/학교 Mac에서 실행이 차단됩니다

**A**: IT 관리자가 Gatekeeper 정책을 엄격하게 설정한 경우입니다.

**해결책**:
1. IT 부서에 요청하여 앱 승인 받기
2. 또는 개인 Mac에서 사용

---

### Q4. 데이터가 저장되지 않습니다

**A**: 다음 확인:

1. **권한 확인**:
   ```bash
   ls -la ~/Documents/
   # ScrapDiary 폴더가 있는지 확인
   ```

2. **수동 저장**:
   - 툴바에서 💾 버튼 클릭

3. **디스크 공간**:
   ```bash
   df -h ~
   # Available 공간 확인
   ```

---

### Q5. 트위터/인스타그램이 표시되지 않습니다

**A**: 

1. **인터넷 연결 확인**
   - SNS 임베드는 인터넷 필요

2. **Fallback 카드 확인**
   - 임베드 로딩 실패 시 자동으로 링크 카드 표시

3. **오프라인 보관**
   - 내보내기 → 안전 모드 사용 권장

---

### Q6. Apple Silicon Mac에서 "Rosetta 설치" 요구

**A**: 잘못된 버전을 다운로드했습니다.

- **arm64.dmg** 버전을 다운로드하세요 (Intel 버전 아님)

---

### Q7. 백업은 어떻게 하나요?

**A**: 자동 및 수동 백업 모두 지원:

**자동 백업**:
- 5초마다 `current.json`에 자동 저장

**수동 백업**:
1. 툴바 → 💾 백업 버튼
2. "새 백업 생성" 클릭
3. `~/Documents/ScrapDiary/backups/` 확인

**복원**:
1. 백업 버튼 → 백업 목록
2. 원하는 백업 선택 → "복원"

## 🔒 프라이버시 및 보안

### ✅ 완전한 로컬 저장
- 모든 데이터는 `~/Documents/ScrapDiary/`에만 저장
- 외부 서버 전송 없음
- 클라우드 동기화 없음

### ✅ 오프라인 작동
- 인터넷 없이 완전히 작동
- SNS 임베드만 인터넷 필요 (선택사항)

### ✅ 데이터 소유권
- JSON 형식으로 저장 (다른 도구로 읽기 가능)
- 언제든 백업 파일 직접 관리 가능

## 📝 알려진 제약사항

### macOS 코드 서명
- 현재 버전은 코드 서명되지 않았습니다

- 새 버전은 다운로드 링크로 제공
- 향후 업데이트에서 추가 예정

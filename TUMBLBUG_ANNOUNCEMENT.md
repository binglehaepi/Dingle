## 🔒 개인정보 보호

**Dingle은 여러분의 프라이버시를 최우선으로 합니다.**

### ✅ Dingle은:
- 모든 데이터를 **사용자 컴퓨터에만** 저장합니다
- 외부 서버로 데이터를 **전송하지 않습니다**
- 사용자를 **추적하지 않습니다**
- 분석 도구(Google Analytics 등)를 **사용하지 않습니다**

### ❌ Dingle은 수집하지 않습니다:
- 이름, 이메일, 전화번호
- 위치 정보
- 사용 패턴
- 다이어리 내용

### 📁 데이터 저장 위치
```
C:\Users\[사용자명]\AppData\Local\dingle
```

앱을 삭제한 후 위 폴더를 삭제하면 모든 데이터가 제거됩니다.

**상세 내용**: GitHub의 `PRIVACY_POLICY.md` 참조

## 🔄 자동 업데이트

**한 번 설치하면 끝!**

1. 앱 실행 시 자동으로 최신 버전 확인
2. 새 버전 발견 시 알림 표시
3. "다운로드" 버튼 클릭
4. 자동 설치 완료! ✨

**데이터는 보존됩니다**: 업데이트해도 다이어리 내용은 그대로 유지됩니다.

## ❓ 자주 묻는 질문

### Q: 무료인가요?
**A**: 네! Dingle은 완전 무료 오픈소스 앱입니다.

### Q: Mac/Linux 버전은 없나요?
**A**: 현재는 Windows만 지원합니다. 향후 추가 예정입니다.

### Q: 데이터는 어디에 저장되나요?
**A**: 사용자 컴퓨터에만 저장됩니다. 클라우드 저장은 없습니다.

### Q: 백업은 어떻게 하나요?
**A**: 설정 → 백업/복원에서 백업 파일을 생성할 수 있습니다.

### Q: 다른 PC로 옮기려면?
**A**: 백업 파일을 USB나 클라우드로 복사한 후, 새 PC에서 복원하세요.

                    Dingle v1.0.0 선물 전달 안내
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

안녕하세요, 빙글해피 팀입니다.

텀블벅 후원에 진심으로 감사드립니다.
후원자님께서 후원해주신 Dingle 디지털 스크랩 다이어리를 전달드립니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📥 다운로드 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 방법 1: 첨부 파일 다운로드 (권장)

첨부된 'Dingle_v1.0.0_Tumblbug_Setup.zip' 파일을 다운로드하신 후,
압축을 해제하여 사용하실 수 있습니다.

압축 파일 내 포함 항목:
- Dingle Setup 1.0.0.exe (설치 파일)
- 설치방법.txt
- 문제해결_가이드.txt
- 자동업데이트_안내.txt
- README.txt
- 커스텀_스티커_추가_가이드.txt (스티커 150종 구매자용)


■ 방법 2: GitHub Release 다운로드

GitHub에서 직접 다운로드하실 수도 있습니다.
이 방법을 이용하시면 자동 업데이트가 원활하게 작동합니다.

다운로드 링크: https://github.com/binglehaepi/Dingle/releases/tag/v1.0.0

- Windows 설치 파일: Dingle Setup 1.0.0.exe
- Windows 휴대용 버전: Dingle 1.0.0.exe

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔧 설치 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 'Dingle Setup 1.0.0.exe' 파일을 실행합니다.

2. Windows Defender 경고가 나타날 수 있습니다.
   → "추가 정보" 클릭 → "실행" 클릭
   (코드 서명 인증서 비용 문제로 경고가 표시되나, 안전한 프로그램입니다)

3. 설치 경로를 선택하고 "설치" 버튼을 클릭합니다.

4. 설치 완료 후 바탕화면 아이콘을 클릭하여 실행합니다.

※ 상세한 설치 방법은 압축 파일 내 '설치방법.txt'를 참고해주시기 바랍니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💻 시스템 요구사항
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 운영체제: Windows 10/11 (64비트)
- RAM: 4GB 이상 권장
- 저장 공간: 200MB 이상
- 인터넷: 임베드 콘텐츠 및 오하아사 기능 이용 시 필요

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔄 자동 업데이트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Dingle은 자동 업데이트 기능을 지원합니다.
앱 실행 시 자동으로 최신 버전을 확인하며, 새 버전 발견 시 알림을 표시합니다.
원클릭으로 최신 버전으로 업데이트하실 수 있습니다.

※ 자세한 내용은 '자동업데이트_안내.txt'를 참고해주시기 바랍니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎨 스티커 150종 추가 구매 후원자님께
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

현재 버전에서는 받으신 스티커 파일을 "사진으로 추가" 기능을 통해
다이어리에 사용하실 수 있습니다.

다음 버전(v1.1.0)에서는 커스텀 스티커 라이브러리 기능이 추가되어
더욱 편리하게 스티커를 관리하고 사용하실 수 있습니다.

자세한 사용 방법은 압축 파일 내 '커스텀_스티커_추가_가이드.txt'를
참고해주시기 바랍니다.

🆘 문의 및 지원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제가 발생하거나 문의사항이 있으신 경우:

1. 압축 파일 내 '문제해결_가이드.txt' 참고
2. GitHub 이슈 제기: https://github.com/binglehaepi/Dingle/issues

GitHub 이슈 제기 시 다음 정보를 포함해주시면 신속한 지원이 가능합니다:
- Windows 버전 (예: Windows 11 23H2)
- Dingle 버전 (v1.0.0)
- 오류 메시지 스크린샷
- 문제 재현 방법

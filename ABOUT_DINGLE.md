> 일본 아날로그 다이어리의 감성을 디지털로 재현한 스크랩북 다이어리

## 📖 Dingle이란?

Dingle은 **디지털 스크랩 다이어리** 애플리케이션입니다.

일본의 아날로그 다이어리 문화에서 영감을 받아, 사진, 스티커, 마스킹테이프로 자유롭게 꾸미는 다이어리의 감성을 디지털 환경에서 재현했습니다. 드래그 앤 드롭으로 직관적으로 조작하며, 나만의 스타일로 매일의 소중한 순간을 기록할 수 있습니다.

---

## ✨ 주요 특징

### 🎨 자유로운 꾸미기

- **사진 배치**: 원하는 위치에 자유롭게 배치
- **스티커**: 회전, 크기 조절 가능
- **마스킹테이프**: 다양한 색상의 테이프로 장식
- **드래그 앤 드롭**: 직관적인 조작 방식

### 📅 다양한 뷰 모드

- **월간 캘린더**: 한 달 전체를 한눈에 확인
- **주간 다이어리**: 일주일을 펼친 책처럼 관리
- **일간 다이어리**: 하루에 집중하여 상세하게 기록

### 🎵 멀티미디어 임베드

- **YouTube**: 영상을 다이어리에 직접 임베드
- **Spotify**: 음악 플레이어 추가
- **SoundCloud**: 좋아하는 트랙 저장
- **Pinterest**: 이미지 콘텐츠 저장
- **Twitter**: 트윗 캡처 및 저장

### 🔮 오하아사 운세

- 매일 업데이트되는 12 별자리 순위
- 더블클릭으로 수동 새로고침
- 귀여운 코코넛 캐릭터 디자인

### 💾 안전한 데이터 관리

- **자동 저장**: 작성 중 실시간 자동 저장
- **백업/복원**: 언제든 이전 상태로 복구 가능
- **로컬 저장**: 모든 데이터는 내 컴퓨터에만 저장
- **개인정보 보호**: 서버 전송 없음, 개인정보 미수집

### 🔄 자동 업데이트

- 앱 실행 시 자동으로 최신 버전 확인
- 원클릭으로 업데이트 가능
- 평생 무료 업데이트 제공

## 🚀 시작하기

### 1. 설치

1. [GitHub Release](https://github.com/binglehaepi/Dingle/releases/latest)에서 `Dingle Setup.exe` 다운로드
2. 설치 파일 실행
3. Windows Defender 경고 시: "추가 정보" → "실행" 클릭
4. 설치 완료 후 바탕화면 아이콘으로 실행

### 2. 첫 다이어리 작성

1. **날짜 선택**: 좌측 달력에서 날짜 클릭
2. **사진 추가**: 드래그 앤 드롭 또는 우클릭 메뉴
3. **꾸미기**: 스티커, 테이프로 자유롭게 장식
4. **저장**: 자동 저장되므로 따로 저장 버튼 없음!

### 3. 추가 기능 활용

- **임베드**: URL 입력하여 YouTube, Spotify 등 추가
- **오하아사**: 우측 상단 코코넛 캐릭터 클릭
- **백업**: 설정 메뉴에서 백업 파일 생성 가능

## 🔐 개인정보 보호

Dingle은 사용자의 개인정보를 최우선으로 보호합니다:

- ✅ **로컬 저장**: 모든 데이터는 사용자의 컴퓨터에만 저장
- ✅ **서버 전송 없음**: 다이어리 내용이 외부로 전송되지 않음
- ✅ **개인정보 미수집**: 이메일, 이름 등 어떠한 개인정보도 수집하지 않음
- ✅ **오픈소스**: 코드가 공개되어 투명성 보장

자세한 내용: [개인정보 보호 정책](PRIVACY_POLICY.md) | [보안 감사 보고서](SECURITY_AUDIT_REPORT.md)
